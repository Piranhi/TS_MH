Use the new image references as the visual north star.

## Direction

The preferred style is a dark fantasy desktop idle dashboard:

- dark charcoal and black-blue background
- aged gold accents
- subtle stone, metal, parchment, and hunter-guild texture
- crisp rectangular panels with thin borders
- dense but readable layouts
- ornamental divider details used sparingly
- high quality item and monster art as focal points
- side navigation with icon + label rows
- top resource bar for energy, gold, renown, level, and prestige

## Strongest Reference

The hunt overview reference is the strongest MVP direction. It communicates idle progress immediately:

- auto hunt status
- target monster
- current area
- rewards per hour
- recent encounters
- drops and materials
- compact combat stats
- active challenges

This should influence the first playable screen.

## Inventory Reference

The inventory reference is useful for later, but MVP inventory should be simpler. Keep:

- gear grid
- selected item detail panel
- clear equip/sell/dismantle actions
- right-side stat summary

Delay:

- many equipment slots
- heavy filtering
- compare flows
- large inventory capacity systems

## UI Rules

- No old frosted-glass blue/purple SaaS look.
- Use gold highlights for active navigation, important resource values, and prestige.
- Keep panels compact and scan-friendly.
- Prefer clear numbers, rates, and timers over explanatory text.
- Use icons for resources and navigation.
- Make the first screen the actual idle dashboard, not a landing page.
- Avoid complex decoration that competes with the game data.

## MVP Screens

Start with:

1. Hunt dashboard.
2. Character/training panel.
3. Simple inventory/equipment panel.
4. Basic settlement/support panel if needed.

The hunt dashboard should carry the identity of the game.
